# TripWay — Guia de Viagens

Projeto desenvolvido para o **Checkpoint 1** da disciplina de Front-End.

O TripWay é um site fictício de guia de viagens criado com HTML puro, aplicando conceitos de estrutura semântica, formulários, navegação multipágina e uso de imagens.

---

## Páginas do Projeto

| Arquivo | Descrição |
|---|---|
| `index.html` | Página inicial com apresentação da empresa e imagem de destaque |
| `destinos.html` | Lista de destinos com nome, descrição e imagem de cada local |
| `pacotes.html` | Pacotes de viagem disponíveis com preço, duração e o que está incluso |
| `contato.html` | Formulário de reserva e informações de contato da empresa |

---

## Tecnologias Utilizadas

- HTML5
- Semântica HTML (`header`, `nav`, `main`, `section`, `footer`)
- Formulários HTML
- Navegação multipágina com caminhos relativos
